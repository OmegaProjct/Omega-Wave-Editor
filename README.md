# Omega Wave Editor

Ein moderner, plattformübergreifender **Desktop‑DAW** (Digital Audio Workstation) für die kreative Audio‑Bearbeitung. Die Anwendung ist in **Electron**, **React** und **TypeScript** gebaut und bietet:

- **Echtzeit‑Lautstärken‑Dragging** (Gain‑Linie) mit sofortiger Rückmeldung über die Web‑Audio‑Engine.
- **ID3‑Tag‑Editor** im Export‑Modal, inkl. automatischer Format‑Synchronisation.
- **VST‑Plugin‑Bridge** (Windows) zum Laden von VST‑Effekten.
- **Automatischer Update‑Check** über GitHub‑Releases.
- **Portable‑Version** (`Omega‑Wave‑Editor‑Portable‑0.2.0.exe`) für den schnellen, installationsfreien Start.

Die aktuelle Version ist **0.2.0** und enthält Cross‑Platform‑Build‑Konfigurationen für Windows, macOS und Linux.
